Request

POST /v1/checkouts

    {
    "recordId": "<record id>",
    "formatId": <format id>"
    }
record id The content provider record id.
format id The content provider record format id.

Response

    {
    "id": "<checkout id>",
    "contentUrls": "<content urls>",
    "expirationDate": "<expiration date>"
    }
checkout id The checkout id. This id should be used as a key when fetching a checkout in getCheckout.
content urls The media content urls array.
expiration date The checkout expiration date in ISO 8601 format.

Example response:

    {
    "id": "123456",
    "contentUrls": ["https://provider.com/ehub-api/v1/records/123456789/content_0", "https://provider.com/ehub-api/v1/records/123456789/content_1"],
    "expirationDate": "2016-01-01T00:00:00Z"
    }
