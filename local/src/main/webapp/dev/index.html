eHUB Development

Axiell eHUB is a media proxy between e-media content providers and end users. Axiell eHUB APIs provide a common interface to several e-media content providers collections. They allow third party applications to integrate these collections to discover and circulate digital media.

Getting started with eHUB

eHUB could serve both as a content provider to any Library System (currently Axiell Arena) and as a content consumer interfacing any other content providers (ex. Overdrive, Ocd, Elib). For this purpose it can be interfaced in the following ways:

  • Through eHUB API, when eHUB acts as a content provider to the Library Systems (server implementation).
  • By adding a new media provider, when it interacts with other media providers (client implementation).
