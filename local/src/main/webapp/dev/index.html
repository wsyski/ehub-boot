eHUB Developers

Axiell eHUB is a media proxy between media vendors and end users. Axiell eHUB APIs provide a common interface to several media vendors' digital collections. They allow third party applications to integrate these collections to discover and circulate digital media.

Getting started with eHUB APIs

  • Read through the eHUB API documentation.
  • Explore the eHUB APIs.
  • Apply for a developer account by contacting your sales representative, see Axiell home page for more information.
  • Sign a contract with one or several Content Providers that are integrated with the eHUB.
