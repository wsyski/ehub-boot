eHUB Development

Axiell eHUB is a media proxy between e-media content providers and end users. Axiell eHUB APIs provide a common interface to several e-media content providers' collections. They allow third party applications to integrate these collections to discover and circulate digital media.

Getting started with eHUB APIs

eHUB could serve both as a content provider to any Library System (currently Axiell Arena) and as a content consumer interfacing any other content providers (ex. Overdrive, Ocd, Elib). For this purpose exposes the following interfaces:

  • The eHUB API, when it acts as a content provider to the Library Systems (server implementation).
  • The eHUB Provider API, when it interfaces other media providers (client implementation).
