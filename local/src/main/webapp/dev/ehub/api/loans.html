Using the Loans API

Using POST and GET HTTP requests, you can use the Loans API to do the following:

  • Create a loan
  • Retrieve an ongoing loan by the eHUB loan ID
  • Retrieve an ongoing loan by the LMS loan ID

Create a loan

Below is the example POST request sent to the v1/loans endpoint.

POST https://ehub.axiell.com/v1/loans
Authorization: eHUB ehub_consumer_id="1", ehub_library_card="909265910", ehub_pin="4447", ehub_signature="GN%2B9mlD70ZER%2Fx3ur7w7HJRgnYU%3D"

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<ehub:pendingLoan contentProviderFormatId="58" contentProviderName="atingo" contentProviderRecordId="9100128260" lmsRecordId="lmsRecordId1" xmlns:ehub="http://ehub.axiell.com"/>

The response includes the URL to the content of the loan plus metadata about the loan.

[Some HTTP Headers]
Content-Type: application/xml; charset=utf-8

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<ehub:readyLoan id="1" xmlns:ehub="http://ehub.axiell.com">
    <contentProviderLoan expirationDate="2014-09-04T09:29:16.622+02:00" id="123456789">
        <streamingContent height="100" url="https://example.com/content" width="100"/>
    </contentProviderLoan>
    <lmsLoan id="lmsLoanId1"/>
</ehub:readyLoan>

Retrieve an ongoing loan by the eHUB loan ID

Below is the example GET request sent to to the v1/loans/{ID} endpoint.

GET https://ehub.axiell.com/v1/loans/123456789
Authorization: eHUB ehub_consumer_id="1", ehub_library_card="909265910", ehub_pin="4447", ehub_signature="GN%2B9mlD70ZER%2Fx3ur7w7HJRgnYU%3D"

The response includes the URL to the content of the loan plus metadata about the loan.

[Some HTTP Headers]
Content-Type: application/xml; charset=utf-8

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<ehub:readyLoan id="1" xmlns:ehub="http://ehub.axiell.com">
    <contentProviderLoan expirationDate="2014-09-04T09:29:16.622+02:00" id="123456789">
        <streamingContent height="100" url="https://example.com/content" width="100"/>
    </contentProviderLoan>
    <lmsLoan id="lmsLoanId1"/>
</ehub:readyLoan>

Retrieve an ongoing loan by the LMS loan ID

Below is the example GET request sent to to the v1/loans?parameters endpoint.

Query Parameters
lmsLoanId The ID of the loan in the Library Management System (LMS). 
GET https://ehub.axiell.com/v1/loans/?lmsLoanId=lmsLoanId1
Authorization: eHUB ehub_consumer_id="1", ehub_library_card="909265910", ehub_pin="4447", ehub_signature="GN%2B9mlD70ZER%2Fx3ur7w7HJRgnYU%3D"

The response includes the URL to the content of the loan plus metadata about the loan.
