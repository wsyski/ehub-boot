  4. getCheckout

Request

GET /v1/checkouts/<checkout id>

checkout id The checkout id.

Response

    {
    "checkoutId": "<checkout id>",
    "contentUrl": "<content url>",
    "expirationDate": "<expiration date>"
    }
checkout id The checkout id.
content url The media content url.
expiration date The checkout expiration date in ISO 8601 format.

Example response:

    {
    "checkoutId": "123456",
    "contentUrl": "https://provider.com/ehub-api/v1/records/123456789/content",
    "expirationDate": "2016-01-01T00:00:00Z"
    }
